# 对中国持普通护照的公民实施免签政策的国家和地区

## ✅ 对中国持普通护照公民免签入境的国家和地区完整名单

（仅限旅游、商务、探亲、过境等短期目的，不含落地签）

> **说明**：
> 
> *   “免签” = 无需提前申请签证，凭护照直接入境。
> *   分为两类：**互免签证**（双方协议） 和 **单方面免签**（对方单方给予）。
> *   **不包含落地签、电子签、需邀请函或特殊条件的“准免签”**（如需美签/申根签才能免签的不算）。
> *   所有信息均来自官方渠道，截至 **2026年1月1日有效**。
