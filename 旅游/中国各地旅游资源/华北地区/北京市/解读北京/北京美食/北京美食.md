# 北京美食  

京菜擅长烤、爆、烧、焖、涮，听起来豪爽，吃起来痛快。北京烤鸭是来京游玩必食的美味。  

北京风味小吃有600多年历史，包括汉民风味小吃、回民风味小吃和宫廷风味小吃等约二三百种。目前，在北京市面上能见到的小吃有：焦圈、豌豆黄、豆面糕（驴打滚）、炸糕、年糕、豆腐脑、茶汤、油炒面、艾窝窝、烤白薯、豆汁、炒肝、馄饨、炒疙瘩等。  

⏩北京烤鸭  
北京名食，它以色泽红艳，肉质细嫩，味道醇厚，肥而不腻的特色，被誉为“天下美味”而驰名中外。北京烤鸭分为挂炉烤鸭和焖炉烤鸭两大流派，全聚德（挂炉）和便宜坊（焖炉）是两派的代表。  

⏩炸酱面  
由黄瓜、香椿、豆芽、青豆、黄豆等菜码，五花肉丁及葱姜制作的炸酱，拌面条而成，味道鲜美。炸酱讲究“小碗干炸”——小火慢熬，使油脂与酱料充分交融，酱香浓郁。  

⏩豆汁  
具有色泽灰绿，豆汁浓醇，味酸且回味微甘的特色。冬、春季流食小吃，尤其是老北京人对它有特殊的偏爱。喝豆汁须配焦圈和酱菜。  

⏩褡裢火烧  
因制作成形后，酷似旧时人们腰带上的“褡裢”，因而得名。皮薄柔软，金黄油亮，味美香鲜，热食最宜。由顺义人姚春宣夫妇于1876年创制。  

⏩炒肝  
以猪肝、肥肠为原料，加入大量蒜末熬煮，最后淋入淀粉水勾芡而成。吃法讲究托着碗底转圈吸溜，常配猪肉大葱包子。  

⏩铜锅涮肉  
传统炭火铜锅，清汤锅底，突出羊肉鲜美。东来顺的涮羊肉制作技艺已被列入国家级非物质文化遗产。  

⏩卤煮火烧  
以猪肠、猪肺、炸豆腐等为主料，炖煮在浓郁老汤中，火烧浸满汤汁。  

⏩爆肚  
分羊爆肚和牛爆肚，羊爆肚有9种不同部位，蘸麻酱、腐乳汁、辣椒油等调料食用，质地鲜嫩，口味香脆。
