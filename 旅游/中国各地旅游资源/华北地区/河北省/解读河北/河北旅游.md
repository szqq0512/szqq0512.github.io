# 河北旅游  
河北精彩推荐  
【山海关】：明长城东端的重要关隘，与嘉峪关并称“天下第一关”。北依燕山，南濒渤海，地势险要，自古为兵家必争之地。主要街道和小巷大部分保留原样，保存众多四合院民居，使古城更加典雅古朴。1961年公布为第一批全国重点文物保护单位。2018年被评为国家5A级旅游景区。国家5A级旅游景区。全国重点文物保护单位（第一批）。  
【北戴河】：一脉青山，山光积翠；一汪碧水，水色含青。北戴河海滨环境优美，风光秀丽。中国四大避暑地之一，首批国家级重点风景名胜区。国家4A级旅游景区。国家级旅游度假区。  
【避暑山庄】：承德市的象征，清代皇帝夏天避暑和处理政务的场所。中国四大名园之一，我国现存规模最大的古典皇家园林。1994年列入世界文化遗产。2007年评为国家5A级旅游景区。建筑布局分为宫殿区和苑景区（湖区、平原区、山区三部分），内有康熙乾隆钦定七十二景。国家5A级旅游景区。世界文化遗产。全国重点文物保护单位。  
【娲皇宫】：俗称“奶奶顶”，传说中女娲抟土造人、炼石补天的地方。整个建筑倚山就势，构筑严谨，巧夺天工。登阁远眺，太行群峰叠翠。全国重点文物保护单位、国家级风景名胜区。2015年10月被评为国家5A级旅游景区。国家5A级旅游景区。全国重点文物保护单位。国家级风景名胜区。  
【宋辽边关地道】：宋，位于永清县、雄县等地。北宋时期修建的大型地下军事防御工程，被誉为“地下长城”。1993年公布为省级文物保护单位。2006年（永清段）列为第六批全国重点文物保护单位；2013年（雄县段）列为第七批全国重点文物保护单位。全国重点文物保护单位（第六批、第七批）。  
【万里长城（明）】：明长城，分布于山海关—万全县、怀来县—沙河县等地。2015年，河北省政府将全省已认定长城公布为省级文物保护单位，明长城部分归入省级文物保护单位“万里长城（明）”。其中山海关段为第一批全国重点文物保护单位。省级文物保护单位（归入“万里长城（明）”）。  
【古长城】：战国—金，分布于承德、张家口、保定等地区。早期长城共387处，归入省级文物保护单位“古长城（战国—金）”。省级文物保护单位（归入“古长城（战国—金”）。  

河北文物旅游  
古为燕赵之地，文物古迹颇丰。承德（1982年首批）、保定（1986年第二批）、邯郸（1994年第三批）、正定（1994年第三批）、山海关（2001年）、蔚县（2018年）6座城市入选国家历史文化名城。自先秦至明代长城多经本省，更有“天下第一关”山海关等重要关隘。有全国最大的皇家园林之一避暑山庄，有气势宏伟、石雕精美的清东陵和清西陵，有以出土金缕玉衣而闻名世界的满城汉墓群，还有南北响堂山、苍岩山桥楼殿等宗教遗存。赵州桥在世界桥梁史上地位显著。沧州铁狮子列入“中国之最”。邯郸是赵国文化的中心，有丛台、回车巷、学步桥等。正定多佛寺，近年以拍摄电视剧《红楼梦》修建荣国府为契机，旅游业大见成效。现代革命史迹有西柏坡革命纪念馆等。自然旅游资源方面，最突出的是秦皇岛、北戴河一直延伸到昌黎黄金海岸的海水浴场；木兰围场冈丘起伏、草茂花繁，为清代皇家秋狝习武的场所。近年涞水野三坡因富有野趣而异军突起，列为国家级风景名胜区。  

石家庄境内的古长城绝大部分是明代长城。当时明长城分为外长城和内长城，外长城始建于明朝初年，东起鸭绿江，西达嘉峪关，即人们常说的万里长城。此后，明王朝为防止蒙古贵族从张家口、大同一带南下，于成化年间开始沿太行山脊修筑长城，形成了内长城。内长城北起张家口市怀来县，向南经张家口市涿鹿县、保定市涞源县等，经石家庄市境内的平山、井陉、赞皇，到达邢台市的内丘、邢台、沙河，直到邯郸的武安市。2015年，河北省政府将全省已认定长城公布为省级文物保护单位，共389处。  

河北吃喝玩乐  
河北的特色风味菜肴有饶阳杂面、唐山蜂蜜麻糖、保定马家老鸡铺卤鸡、藁城宫面。河北还是一个出产名酒的地方，长城干白葡萄酒果香浓郁、色泽清亮，是国内葡萄酒中的上品。河北梆子、老调、皮影、丝弦等戏曲饶有特色。
