# 喀什地区  
喀什最有名的便是纵横交错、布局灵活的老城，特色是以伊斯兰建筑风格为主体的迷宫式街区。老城的主要街道都通向艾提尕尔清真寺，在星期五早晨，可以听到寺中召唤教徒做礼拜的声音。

⏩莎车无人机报备指南  
🔸地点：莎车县公安局（和喀什不同，不是交通巡警大队）  
1.跟门卫说明来意，他会打电话联系工作人员，然后由工作人员带领进入。三楼315室。  
2.填写表格，如果有喀什的无人机报备表格，给他看一下，他会对比着打印一张。  
3.盖章后即可。  
🔸飞机解禁：  
1.在大疆官网申请解禁  
2.填写个人认证信息，与报备表格中联系人要是同一人。  
3.选择“解禁自定义区域”，因为飞行半径只有500米，经纬度不一定能包含你想拍的地方，所以最好用手绘的方式圈出老城区域。  
4.提交申请，大概15分钟左右能够审核完毕。  
🔸导入飞行证书  
1.解禁成功之后，需要打开无人机，进行证书导入。此步骤大疆有操作说明。  
2.解禁成功后，禁飞区提示消失，可以起飞了  
‼️特别提醒：与喀什直接在老城区巡警大队报备不同，因为莎车是在县公安局报备的，所以还需要再在打算飞行区域的区派出所进行备案，他们会复印一份申请表，进行备案。

* 【慕士塔格冰川公园】：徒步比较辛苦，正常到4号冰川只能看到底部的冰舌，但毕竟意义非凡。4号冰川。门票：90元。开放时间：10:00—20:00（12:00—18:30）。走一公里到最高处，海拔4680m（可骑马前往，淡季单程100块）。游览时间来回约需三个小时。*中国原生态景点。*  
* 【塔什库尔干塔吉克自治县博物馆】：免费。塔什库尔干县慕士塔格路112号。  
* 【塔什库尔干县塔吉克历史博物馆】：*国家2A级旅游景区。*  
* 【红其拉甫口岸】：红其拉甫口岸早在1000多年前就是古代丝绸之路上一个重要的关隘，风光壮美，但环境恶劣，素有“死亡山谷”之称。到红其拉甫口岸，游客一般都会到中巴7号界碑处拍照留影，2009年新建的国门就在此处，威严肃穆。  
* 【红其拉甫国门景区】：门票34元/人次、区间车票5.5元。统一在帕米尔游客中心购票，统一乘坐区间车进入景区，禁止私家车前往。需提前一天预约（帕米尔旅游景区公众号，08:00至16:00）。景区发车时间及班次：每天10:30和16:30，一天两班次。景区游玩路线：塔县帕米尔旅游景区游客服务中心—紫花牧场服务区（卫生间）—前哨班—红其拉甫国门—紫花牧场服务区（卫生间）—塔县帕米尔旅游景区游客服务中心。旅游券40元/人，区间车160元/人。海拔5200米。咨询热线：0998–3493328。*国家4A级旅游景区。*  
* 【公主堡古城遗址】：公主堡位于塔士库尔干县城以南约70公里处，在古丝绸之路咽喉地段卡拉其峡谷的一座海拔4000多米的高山上，是我国目前所知的最高的古代城堡之一。在帕米尔高原，公主堡圣洁而又崇高，现存的古堡遗址前临奔腾咆哮的塔什库尔干河，后倚高耸的皮斯岭达坂，景色绝美。年代：唐。达布达尔乡卡拉其甫西北，卡斯库尔干奥夫所在的塔什库尔干河谷西侧陡山上。*全国重点文物保护单位。*  
* 【吉尔赞喀勒墓地】：年代：东周。提孜那甫乡曲什曼村北，塔什库尔干河西岸。*全国重点文物保护单位。*
