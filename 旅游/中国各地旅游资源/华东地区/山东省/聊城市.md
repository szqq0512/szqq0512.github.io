# 聊城市  
🚩国家历史文化名城。  
🏅中国优秀旅游城市。  
  
📋聊城市地处经济发达的山东省，居鲁西，临河南、河北，位于华东、华北、华中三大行政区交界处。代表中国商业文明的京杭大运河和代表农业文明的黄河在此交汇，贯穿中国南北的京九铁路和连接祖国东西的胶济邯铁路及高速公路在此相交形成“黄金大十字”。 
  
📋聊城因古有聊河而得名（一说因春秋时期曾称聊国故名）。古称东昌，历史悠久，人文景观和自然景观极为丰富。风景如画的东昌湖，面积达4.2平方公里，聊城古城座落在湖中央，形成了“城中有湖，湖中有城，城湖一体”的独特的古城格局和风貌，是著名的中国北方水城，被誉为“东方威尼斯”。全市主要旅游资源聊城山陕会馆，临清舍利塔、清真寺，东阿曹植墓，阳谷景阳冈等。聊城也是人杰地灵的地方，有著名学者傅斯年、季羡林、国画大师李苦禅、民族英雄张自忠和党的好干部孔繁森等名人倍出。

🛣️聊城市“望岳湖”旅游风景道，是聊城市南北向的主要交通干线和旅游轴线，分布有凤凰苑植物园、省级文保傅氏家族墓、3处3A级旅游景区、运河样板段、徒骇河休闲观光带、“一校三馆”、望岳湖、望岳湖文体公园、望岳湖星空乐园、盛景酒店、休闲采摘园等文旅资源。（包括江北水城旅游度假区段、东昌府区段）
  
🍴美食：  
现代大型宴席--”鲁西文化宴“总计一百单八款菜肴。或依据历史事件，或凭借民间传说，或利用地方特产，或源于古刹丛林，或挖掘古馆遗菜，或革新传统名肴，皆出之有据，讲之有趣，观之悦目，食之惬意。著名的有：济美酱菜、老王寨驴肉、八批果子、聊城呱嗒、托板豆腐、魏氏熏鸡。其中老王寨驴肉，是高唐传统名吃。其驴肉加工已有300多年的历史。因当地人俗称驴为鬼，故称驴肉为“鬼子肉”。老王寨“鬼子肉”加工方法独特，药料齐全，工艺精湛。其产品呈酱紫色，清香鲜美，香而不腻，烂而不散，具有活血降压、滋补益身之作用。驴鞭称作“金钱肉”， 更具补肾壮阳之功效。当地人招待客人时有“无驴肉不成宴”之说。

🧭行程规划：  
⏩第1天：聊城市区文化探索  
🔸上午：首先前往中华水上古城，以光岳楼为中心开始您的旅程，感受古城的历史韵味。参观附近的山陕会馆，了解古代商业文化和精美的建筑艺术。  
🔸中午：在古城内或附近品尝地道的鲁菜，如把子肉、甏肉干饭等地方特色美食。  
🔸下午：参观运河博物馆，了解京杭大运河的历史与聊城在其中的重要地位。之后，乘坐游船游览东昌湖，享受湖光水色与城市风光的完美融合。  
🔸傍晚：前往滨河大道散步，享受夕阳下的河畔风光，或在湖边的咖啡厅小憩。  
⏩第2天：阳谷县水浒文化之旅  
🔸上午：驱车前往阳谷县，首先探访狮子楼，感受《水浒传》中西门庆与武松的故事背景。  
🔸中午：在阳谷品尝当地特色菜肴，比如鲁西黄牛的美味料理。  
🔸下午：前往景阳冈，追寻武松打虎的传奇足迹，体验水浒文化主题公园的风采。  
🔸傍晚：返回聊城市区，晚上可选择在市区内的特色餐馆或小吃街继续品尝美食。  
⏩第3天：自然风光与文化遗迹  
🔸上午：前往江北水城旅游度假区，体验不同寻常的水上娱乐项目，或是在度假区内的自然风光中漫步。  
🔸中午：在度假区内享用午餐，尝试更多的地方特色美食。  
🔸下午：根据个人兴趣，选择参观孔繁森纪念馆、中国运河文化博物馆等文化景点，或是继续在度假区享受休闲时光。  
🔸晚上：如果体力允许，可以在结束之前逛逛聊城的夜市，如朱雀街夜市，体验当地夜生活，购买一些手信作为纪念。

🚩革命文物保护利用片区（冀鲁豫片区）。  
  
* 【聊城古城】（东昌古城）：古城特色有：光岳楼、山陕会馆、东昌湖等。*中国原生态景点。*  
* 【光岳楼】：位于聊城故城中央。高楼凌空，巍峨壮丽，气势非凡，为鲁西一大名胜。它始建于明洪武七年（1374年）。当时，东昌卫守御指挥佥事陈镛在重修城垣时，为“严更漏，窥敌望远”，利用剩余木料建造而成。故始称“余木楼”，后亦称“鼓楼”、“东昌楼”。明弘治九年（1496年），考功员外郎李赞到聊城，见此楼高壮极目，天下所无，但及今百年，尚寞落无名，因与当时东昌府太守金天锡商定，命名“光岳楼”，“取其近鲁有光于岱岳”。该楼是一座由宋元向明清过渡的代表建筑，系我国现存明代楼阁中最大的一座。全楼有112个台阶、192根金柱、200余斗拱。年代：明。东昌府区古城中心。*中国大运河遗产点，世界文化遗产。全国重点文物保护单位。*  
* 【范筑先烈士纪念馆】：庭院中心有一通用黑色大理石雕成的石碑，正面镌刻着邓小平同志亲笔题写的“民族英雄范筑先殉国处”，碑阴刻有范将军传略。纪念馆大厅檐下正中悬挂一方由梁漱溟先生题写的“范筑先烈士纪念馆”木刻横匾。范筑先（1882-1938）河北省馆陶县人。抗日战争爆发时，范任山东省第六区行政督察专员兼保安司令。日军进犯山东，范发表誓死不过黄河的皓电，扩大抗日武装，建立地方抗日政权，与日军战斗80余次，收复23个县的大片土地，1938年11月，日军侵犯聊城，范指挥抵抗，于激战中壮烈殉国。  
* 【海源阁】：由清代著名藏书家杨以增建于清道光20年（1840）。海源阁原为三间硬山南向二层楼，楼前有长廊式读书亭两座。海源阁藏书楼是我国历史上最著名的私人藏书楼之一，累计藏书22万卷，其中宋元珍本逾万卷。  
* 【傅斯年陈列馆】：1994年程思远副委员长专门为傅斯年陈列馆寄来了题词：“傅公高风亮节，足为后世楷模”。季羡林先生为傅斯年陈列馆题写了匾额。傅斯年（1896－1950年），字孟真，聊城人，清朝开国状元傅以渐第七世孙，中国现代史上著名的史学家。傅斯年先生学贯中西，博古通今，对历史学、考古学、语言学、教育学等造诣精深、贡献巨大，被誉为学术界的“通才”。傅斯年先生曾为国共两党和谈积奔走。

* 【江北水城旅游度假区】：姜堤村。接待联系：李平15863503618。  
* 【聊城山陕会馆】：它是由山西、陕西商人合建的一处神庙与会馆相结合的建筑群。保留至今的有山门、戏楼、夹楼、鼓钟2楼、南北看楼、南北碑亭、飨亭、南中北3殿游廊、春秋阁、望楼等亭台楼阁160余间。院中两尊高约2.8米的石狮子，造型雄伟，比例准确，刀法娴熟，刻工精细，生动地再现了狮子的威武凶猛，是难得的艺术珍品。两株古槐，老干苍劲，丫杈盘绕，更给古建筑增添了古色古香。年代：清。东昌府区双街55号。*中国大运河遗产点，世界文化遗产。全国重点文物保护单位。*
