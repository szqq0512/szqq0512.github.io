# 丽江市  
  
🚩国家历史文化名城（专项督察县域单元：古城区）。  
🏅中国优秀旅游城市。  
  
📋丽江市是云南省辖地级市，位于云南省西北部，地处云贵高原与青藏高原的连接部位，地势西北高而东南低，最高点为玉龙雪山主峰，海拔一般都在2000米以上。北连迪庆藏族自治州，南接大理白族自治州，西邻怒江傈僳族自治州，东与四川凉山彝自治州和攀枝花市接壤，距昆明市527公里。总面积2.06万平方公里，辖1个市辖区（古城区），4个县（玉龙、永胜、华坪、宁浪），其中玉龙县为丽江郊区，其它各县距市有一定距离。  
  
📋丽江市自南宋晚末年算起已有800多年历史。丽江座落在三江分流之处，分流经古城。在19世纪晚期出入于云南省的一名美国传教士曾经写过一本书，自此，丽江方为西方人士所知。然而，在中国于20世纪80年代允许国外游客来华后，丽江才真正开始吸引大量的游客。为了满足游客的需求，当地人开始开办了餐厅和咖啡馆。1996年的丽江地震再次让全世界的目光投向了这座小城。国人也非常向往那边悠闲的生活状态和超赞的自然环境，每年都有越来越多的人背上包踏上丽江的土地。  
  
📋2016年末，全市常住人口128.5万人。除汉族外，共有12个世居少数民族。  
 
📋旅游资源丰富，有世界文化遗产丽江古城、玉龙雪山、虎跳峡、老君山、束古镇等知名旅游景点。

🧭快速攻略：   
⏩第一天：丽江古城探索  
🔸上午：抵达丽江后，先入住酒店或客栈，建议选择古城外或靠近城门的位置，以便出入方便。稍作休息后，开始探索【丽江古城】（世界文化遗产），逛逛四方街，感受古城的热闹与古朴。  
🔸午餐：在古城内品尝地道的纳西族美食，如鸡豆凉粉、纳西烤鱼。  
🔸下午：参观木府，了解纳西族土司的历史；之后可以爬上【狮子山】，俯瞰整个丽江古城的全景，以及远处的玉龙雪山。  
🔸傍晚至夜晚：在古城内随意漫步，体验酒吧街的夜生活，但请注意安全，避免深夜单独行动。  
⏩第二天：玉龙雪山一日游  
🔸上午：提前预定玉龙雪山的门票及索道票，参加早上的旅行团或自行前往。乘坐索道上山，途中欣赏雪山壮丽景色。  
🔸中午：在雪山脚下的蓝月谷附近用餐，这里风景如画，适合拍照留念。  
🔸下午：继续游览蓝月谷，体验高山湖泊的静谧与美丽。之后返回丽江古城，视体力情况决定是否参加晚上的民族歌舞表演，如《印象丽江》。  
⏩第三天：束河古镇与拉市海    
🔸上午：前往【束河古镇】，相比丽江古城更为宁静，适合慢慢闲逛，享受悠闲时光。  
🔸午餐：在束河古镇尝试当地特色菜肴。  
🔸下午：前往【拉市海】，体验骑马（需注意安全，选择正规马场）或徒步，欣赏湿地风光，如果季节合适，还可以看到候鸟。  
🔸傍晚：返回丽江，根据个人喜好选择在古城内继续探索或休息。    
  
🧭徒步线路：  
⏩玉龙雪山脚下的蓝月谷徒步  
🔸简介：蓝月谷由四个湖泊组成，分别是玉液湖、镜潭湖、蓝月湖、听涛湖，位于玉龙雪山脚下，环境清幽，风景如画。徒步环绕这些湖泊，可以近距离感受水的清澈与雪山的壮丽，整个行程大约需要一到两个小时。  
🔸特点：适合所有年龄段，无需专业装备，可选择步行或乘坐电瓶车游览。  
⏩玉龙雪山徒步  
🔸简介：虽然玉龙雪山主峰攀登需要专业设备和向导，但山脚下有许多较易行走的徒步路径，如从雪山脚下的牦牛坪到云杉坪，沿途可欣赏到高山草甸、原始森林以及雪山景观。  
🔸特点：海拔较高，需注意高反，适合有一定体能基础的徒步者。  
⏩虎跳峡高路徒步  
🔸简介：从虎跳峡桥头出发，沿高路徒步至Halfway或Tina's Guesthouse，这条线路是世界级的徒步路线，以其壮观的峡谷景色著称。  
🔸特点：较为挑战性，需要良好的体力和适当的装备，沿途有住宿点供休息。  
⏩巴拉神山徒步  
🔸简介：这是一条3日穿越的线路，包括高山露营，需要通过云行户外等机构报名参加。  
🔸特点：适合寻求冒险和深度体验自然的徒步者，需要良好的体能和露营装备。  
⏩黎明老君山徒步  
🔸简介：黎明红石街到格拉丹草原，这是一条充满挑战同时景观绝美的线路，海拔高，适合有经验的徒步者。  
🔸特点：景观指数高，体力要求较高，可以欣赏到独特的丹霞地貌和高山草原。  
⏩束河至文海徒步  
🔸简介：这条路线全长约13km，从束河古镇出发，经过红山村，沿途自然风光优美，人迹罕至，适合喜欢安静探索的徒步者。  
🔸特点：较为小众，路况以土路为主，适合结伴而行，注意携带足够补给和导航工具。

## 玉龙纳西族自治县  
🌐所辖行政区：黄山镇（政府驻地）、石鼓镇、巨甸镇、白沙镇、拉市镇、奉科镇、鸣音镇、太安乡、龙蟠乡、黎明傈僳族乡、鲁甸乡、塔城乡、大具乡、宝山乡、石头白族乡、九河白族乡。  
🚩革命文物保护利用片区（长征片区红二方面军）。  
🏅中国旅游强县。  
  
⏩国家级非物质文化遗产代表性项目保护单位：  
🔸三多节：申报：云南省丽江市玉龙纳西族自治县，保护单位：玉龙纳西族自治县非物质文化遗产保护中心。  
  
* 【玉龙雪山景区】：是中国最南的雪山，也是横断山脉的沙鲁里山南段的名山，北半球最南的大雪山。玉龙雪山山脚、东八谷、文笔海，9-10月格桑花。玉峰寺，玉龙雪山山脚，2月-3月山茶花。*国家5A级旅游景区。中国国家地质公园。旅游安全保障最佳景区。云南100个最美观景拍摄点。*  
* 【蓝月谷】：其前身为早前人们所熟知的“白水河”。在晴天时，水的颜色是蓝色的，而且山谷呈月牙形，远看就像一轮蓝色的月亮镶嵌在玉龙雪山脚下，所以名叫蓝月谷。而白水河这个名字是因为湖底的泥巴是白色的，下雨时水会变成白色，所以又叫白水河。蓝月谷中的河水在流淌过程中因受山体阻挡，形成了四个较大的水面，人称“玉液”湖、“镜潭”湖、“蓝月”湖和“听涛”湖。*云南100个最美观景拍摄点（E 100°13′59″ N 27°6′59″）。  * 
* 【九子海】：进玉龙雪山深处，折往东迂回蜿蜒盘旋七八公里，崇山峻岭中的一个盆地里，九子海就明媚青葱的出现在眼前。每年5月至11月，漫山遍野的野花盛开，成了名符其实的花海。  
* 【太安乡】：6月-七月，洋芋花（土豆花）。八月-九月，油菜花。  
* 【千龟山】：在黎明村的西南部，存在有不少红色岩石表面风化形成的美丽的龟裂状构造山体，其中一座山坡形如千万只小乌龟又组成一只大乌龟，排列得自然而且有序，仿佛向着太阳升起的东方正在不停地行进着，堪称黎明丹霞风光中的一绝，美仑而美奂。这就是黎明千龟山的神奇，它是我国迄今为止发现的面积最大的、海拔最高的一片十分神奇丹霞地貌区。*云南100个最美观景拍摄点（E 99°40′48″ N 27°2′31″）。*  
* 【九十九龙潭】：在老君山主脊线北东侧的山地坳里，有湖泊、沼泽数十个，沿溪流成串分布，被称为“九十九龙潭”。在到九十九龙潭的途中，你会看到九块奇美的巨石，它们形态各异、厚凝神秘，当地的人们把它们称为“九子石”。九十九龙潭三潭互依，所以它又被称为“三眼潭”。*云南100个最美观景拍摄点（E 100°12′59″ N 26°51′0″）。*  
* 【石鼓镇】：石鼓，是历代兵家必争的战略要地，因镇上有一面汉白玉雕刻的鼓状石碑而得名，石碑直径15米，厚0.7米，是目前丽江所发现的年代最早的石碑之一。*全国特色景观旅游名镇（村）。云南100个最美观景拍摄点（E 100°2′42″ N 26°54′21″）。*  
* 【长江第一湾】：三月份最美，望城坡村看油菜花和一弯。从丽江进石鼓镇的353国道刚进镇有个一湾饭店，从那店的后山爬上去就能看见最佳的第一湾景色（5-15元）。
* 【红军长征过丽江纪念馆】：免费。丽江市玉龙县石鼓镇。  
* 【石鼓镇大新村委会竹园村】：*中国传统村落。*
