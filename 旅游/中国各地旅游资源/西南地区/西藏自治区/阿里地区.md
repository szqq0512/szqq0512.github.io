# 阿里地区  
📋面积34.5平方公里，平均海拔4500米以上的阿里地区，又有“世界屋脊的屋脊”“高原上的高原”之称，它东西长约700公里，南北宽约680公里，是西藏乃至全国面向南亚开放的桥头堡。  
如果你想远离都市生活，寻找苍茫原始的自然力量，就来阿里吧。阿里境内山系聚集，甚至有“万山之祖、万水之源”的美名。单是巍峨耸立的冈仁波齐和碧水无波的玛旁雍错，就足以震慑天地。而正是在这一片原始荒凉中，诞生了象雄王国和古格王国，成为古西藏文明闪耀的明珠，记得去世界闻名的古格王国遗址和扎达土林看看，在碧空雪山下，触摸古老的西藏脉络。  
  
阿里地区（威利：mnga' ris sa gnas；藏语拼音：Ngari Sanai），位于中国西藏自治区的西部，地处青藏高原北部——羌塘高原核心地带。东起唐古拉山脉以西的杂美山，与那曲地区相连；西及西南抵喜玛拉雅山西段，与印度、尼泊尔及克什米尔地区毗邻；南连冈底斯山中段，临日喀则地区仲巴县、萨嘎县；北倚昆仑山脉南麓，与中国的新疆维吾尔自治区相邻。全区面积30多万平方公里。平均海拔4500米，共辖7个县，人口仅6万，是世界上人口密度最小的地区。阿里是宗教信仰之乡。从古远时代起，这里就是印度教、耆那教、西藏苯教、佛教等尊为“圣灵之所在”的“世界中心”。
  
阿里是喜马拉雅山脉、冈底斯山脉、喀喇昆仑山脉和昆仑山脉相聚的地方，被称之为“万山之祖”。这里又是境内外几条著名江河的发源地，故又称为“百川之源”。阿里地貌以冰雪、山岩和湖泊为其特征，历史上曾把这种特征概括为阿里三围，即冰雪围绕的“普兰”，岩石围绕的“古格”和湖泊围绕的“玛宇”。
  
阿里地区蕴藏着丰富的地热能，有狮泉河──雅鲁藏布江地热带，昆仑山──可可西里地热带，冈底斯山──念青唐古拉山地热带。
  
阿里地区山峦连绵起伏，原野辽阔无际。拥有畜牧业、矿业，可利用草场面积0.186亿公顷，矿产资源已探明的有30多种，矿点254处，其中有盐、硼、铝、芒硝、铬、银、金等。
  
阿里具有地域辽阔，资源丰富的优势，硼矿储量极为丰富，其中盐湖沉积型优质矿已探明储量就达97万吨；阿里独特神奇的自然风光和人文景观吸引了越多越多的国内外游客。现在阿里普兰口岸已定为国家二级口岸，扎达县的什布奇列为传统民间互市贸易点，为阿里地区边贸的发展提供了有利的条件。
  
现今阿里的政治、经济、文化中心狮泉河镇位于印度河上游的狮泉河北岸和新藏、拉狮公路交点上、它不仅是阿里地区行署驻地，更是西藏自治区和新疆维吾尔自治区之间重要的交通连接中心、物资集散地和文化交流支点。    
  
⏩四条从冈仁波齐附近发源的圣河：马甲藏布（孔雀河，普兰县境内）、当却藏布（马泉河，日喀则地区与阿里地区的行政界线）、森格藏布（狮泉河），朗钦藏布（象泉河，流域遍布札达县）。  
  
狮泉河又名森格藏布，系印度河的上游。途经革吉、狮泉河镇等地，在扎西岗与支流噶尔藏布汇合。出境进入克什米尔后称印度河。是西藏降雨量最少的地区，年降水量仅在50-80毫米之间。沿岸以宽谷地形为主，有沼泽和温泉分布。温泉多为火成岩裂隙中涌出的潜水，不但补给了河流，也提高了河水的温度，使一些河段冬季很少封冻。

* 【札达土林–土林大峡谷】（札达土林国家地质公园）：是世界上最典型、分布面积最大的第三系地层风化形成的土林。以托林镇为中心，沿象泉河两岸，东西长达175公里、南北延展45公里的扎达土林地址公园，堪称地质爱好者的心仪之地，千姿百态的景致也博得旅行者的青睐。游览土林的最佳方式是自驾，常规的线路是前往茅刺沟，穿行于565国道，从札达县往返巴尔兵站。这条线路目前路况良好，路面都铺设了柏油，旅行者在车上就可以欣赏到光影创造出的土林风光大片。玛朗景区（不收费）。有收费和不收费的观景台。*西藏100个最美观景拍摄点（E 79°47′59″ N 31°30′0″）。*  
* 【札达土林霞义沟】：札达县城外围的山坡上，土林到处都是，可从县城徒步前往。往来札达时，在途中停车，就可看到大峡谷般的景色。如果往来于札达老路，要经过“土林沟”，景色更好。*西藏100个最美观景拍摄点（E 79°47′59″ N 31°30′0″）。*  
* 【托林镇扎布让村】：*全国乡村旅游重点村。*  
* 【托林寺】：位于札达县托林镇的这座寺庙远看庄严肃穆，如今的规模只有巅峰时期的十分之一。996年，古格国王益西沃与大译师仁钦桑布仿照“藏传佛教第一寺”桑耶寺，建起了这座寺庙。“托林”意为“飞翔于空中永不坠落”，有赖于古格王朝大力弘扬佛法，托林寺逐渐成为当时的佛教中心。如今在佛塔中，还能看到一些源于古格王朝时期的壁画残迹，而寺内拉康嘎波（白殿）和集会殿这两座大殿中的壁画保存得相对完好，近期还经过了修复。需要注意的是，所有壁画都禁止拍照。若想进入殿内观看壁画，记得保留好门票，入殿前仍需查票。始建于北宋时期，是古格王国（公元10–17世纪）在阿里地区建造的第一座佛寺。保存较好的只有三大殿和一座佛塔。年代：宋。县城西北象泉河畔。*全国重点文物保护单位。国家3A级旅游景区。西藏100个最美观景拍摄点（E 79°47′59″ N 31°27′59″）。*

* 【古格王国遗址】（古格王朝遗址）：荒凉、雄壮，札达县札布让区2公里外的古格王国遗址，将这两个标签体现得淋漓尽致。昔日的古格王国城堡，见证了王朝的黄金年代，历经扩建与复兴，最终于17世纪30年代被彻底废弃。从山脚到山顶的300多米间，交错分布着民居、佛塔等建筑500余座、近900个孔洞窟，遗址内还有5座保存较为完好的寺庙和殿堂。在殿堂内部，保存着经过修复的15世纪和16世纪的壁画。沿着山路一路游览至山顶，现存的王宫区院墙残破，夏宫、冬宫和坛城殿还保留着完整的躯壳。古格王国都城遗址建筑在彩绘、泥塑、雕塑艺术等方面都具有很高的价值，是古格王国时期遗留下来的完整作品，也是古格建筑及艺术的代表。在古格王国遗址北边的山沟里，还坐落着传说中的干尸洞，在洞口能看到杂物和一些人骨。年代：约为10世纪前后。县城西20千米扎布让村。*全国重点文物保护单位。中国原生态景点。西藏100个最美观景拍摄点（E 79°40′19″ N 31°27′41″）。*

* 【藏尸洞】：古格王朝的最后遗迹。位于古格故城北面600多米的山沟里。可以不买门票。要爬上一个小洞口（编号1-3-1-30），有三个洞室，里面味道很冲，只能洞口看一下（不建议去看）。
* 【东嘎·皮央洞窟群】：东嘎、皮央是这里两个邻近的村子的名字。遗址分部在东嘎村的部分，是一组石窟群，是中国发现的规模最大的佛教古窟遗址；遗址分布在皮央村的部分，是一处由寺院、城堡、石窟和塔林组成的大型遗址，属于古格王朝仁钦桑布时期所建的八大寺之一，建于10世纪，曾一度是古格王国重要的文化中心。石窟壁画是用矿物颜料绘制的，因此色泽完好。*西藏100个最美观景拍摄点（E 79°49′59″ N 31°39′59″）。*  
* 【皮央和东嘎遗址】：年代：宋至明。东嘎乡皮央村、东嘎村。皮央有上千个洞窟，有“第二敦煌”之称。皮央寺内有精美壁画。*全国重点文物保护单位。*  
* 【皮央东嘎洞窟遗址】：札达县境内相距不远的皮央村和东嘎村，坐落着规模宏大的石窟遗迹。皮央东洞窟遗址主要是由东嘎扎西曲林寺遗址和皮央石窟群组成，包括石窟、壁画、岩画、佛寺与其他建筑遗迹等。皮央东嘎遗址具体创建的年代尚不明晰，但据考证，这里曾是古格王国时期的政治和文化要地。目前，遗址中的石窟总数近千座。皮央石窟的前山和后山共划分为三个区山顶也分为一个区;东嘎石窟群则共分为五个区。来到此处，山坡上密密麻麻开凿的洞窟数不胜数，不过，仍留有壁画且供游客参观的洞窟一共只有五六个。  
> 门票：50元   
* 【曲龙遗址】：位于达巴乡曲龙村，在吐蕃统一时期的7世纪中叶修建，为吐蕃六茹之一的象雄东岱的政治中心。遗址名称意为“大鹏栖息的地方”。*西藏100个最美观景拍摄点（E 79°54′59″ N 31°12′59″）。*  
* 【穹隆银城】（琼隆银城）：是西藏古象雄王国的都城，位于海拔4400米的卡尔东山顶，面积10余万平方米。在阿里这片开阔的土地上，曾孕育出两种古文化，除了如今已被较多人了解的古格文化，还有古象雄文明，而根据学术界的主流观点，穹窿银城正是7世纪古象雄王国的都城。如今的穹降影城遗址，分布在一大片黄土山坡之上，经过多年的考古工作，城中的生活、军事、祭祀场所一一明晰。这里暂未进行现代化的旅游开发，原生环境吸引着诸多渴望探索野性的旅行者。留存的一部分洞内曾被居民当作住处，仔细看，还能发现墙壁被烟火熏过的痕迹。札达县曲龙村西南附近。*国家3A级旅游景区。*

## 措勤县  
该县地处羌塘高原大湖盆地带，属高原丘陵型和高原宽滩型地貌。县境內山峦疊嶂，多为东西走向。整个地势四周高，中部低。属高原亚寒带干旱气候区。光照充足，辐射強，冬春严寒，多风沙，无霜期短。气侯干燥雨水少。年降水量不足100毫米。常见的自然灾害有雪灾、旱灾、风沙、寒流等。措勤县境內有广阔的草原，为纯牧业县。主要饲养牛、黃牛、马、绵羊、山羊等。境內江河、湖泊眾多，鱼类资源较为丰富。但当地居民因习俗和宗教方面的影响，渔业资源尚未开发利用。    
  
措勤县城基本上是拉萨到狮泉河公路的中间点。在这里前往拉萨或狮泉河的卡车都容易找到，价钱一般为150-200元之间。坐卡车前往改则，50元/人。“西藏阿里藏羚羊旅游运输公司”在措勤的联络点是“友谊餐厅”，联系电话0897-2612280。卧铺车票价为300元左右。
住宿：人武部招待所（推荐）、扎日南招待所或者县政府招待所（对外叫高原旅馆）。  
其他：措勤有邮局。

* 【扎日南木错】： 扎日南木错（Zhari Namco）亦称塔热错。属东西向构造断陷湖，咸水湖。扎日南木错是阿里最大、西藏第三大的湖泊，即使如此，历史上的扎日南木错依然比今天大了2倍有余。站在湖边的观景台，居高临下，远眺雪山，俯看草原，透彻的白、绿、蓝交映，标致的高原景色跃然眼底。若前往北岸的木若峰观景台，还能看到蜿蜒深入水面的沙丘，平添了秀美之意。建议前往扎日南木错的游客们选择附近的村落人住，在观景台附近扎营虽然离美景更近，但风险也较高。措勤通往扎日南木错木若峰观景台的必经之路附近设置了门票站，会收取120元的过路费。在湖的正北濒湖而立的木诺山孤峰，由两、三亿年前形成于海底的黑色石灰岩构成，海拔5174米，高出湖面500多米，攀上山崖，扎日南木错尽收眼底，这里是观赏湖光山色最好的景点。措勤县磁石乡格玛村。*国家3A级旅游景区。西藏100个最美观景拍摄点（E 85°36′59″ N 30°55′59″）。*
